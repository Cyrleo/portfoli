# À propos de cette certification

La certification **AWS Certified Solutions Architect - Associate** valide mon expertise dans la conception et le déploiement de systèmes évolutifs, hautement disponibles et tolérants aux pannes sur AWS.

## Compétences validées

- Conception d'architectures cloud sécurisées et robustes
- Déploiement d'applications sur AWS
- Migration d'applications complexes vers AWS
- Optimisation des coûts sur AWS
- Définition d'une architecture cloud répondant aux exigences métier

## Domaines couverts

### Architecture résiliente
- Conception de systèmes découplés
- Choix des solutions de stockage appropriées
- Conception d'architectures multi-niveaux
- Conception de solutions hautement disponibles et/ou tolérantes aux pannes

### Architecture performante
- Choix de solutions de stockage performantes
- Choix de solutions de calcul et de base de données performantes
- Sélection d'options réseau appropriées
- Choix de solutions de mise en cache appropriées

### Architecture sécurisée
- Conception d'accès sécurisé aux ressources AWS
- Conception de charges de travail et d'applications sécurisées
- Définition de contrôles de sécurité appropriés

### Architecture optimisée en termes de coûts
- Conception de solutions de stockage rentables
- Conception de solutions de calcul rentables

## Préparation

Pour obtenir cette certification, j'ai :
- Suivi les formations officielles AWS
- Réalisé des projets pratiques sur AWS
- Étudié les architectures de référence AWS
- Passé des examens blancs

## Validité

Cette certification est valable 3 ans et nécessite une recertification pour maintenir le statut actif.
